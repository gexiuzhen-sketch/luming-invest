## 🎯 评分逻辑详解

### 综合评分公式
```
基础分 = 50

+/- 涨跌幅影响
  +3%以上: +5分
  +1%以上: +2分
  -1%以下: -2分
  -3%以下: -5分

+/- RSI影响
  超卖(<30): +5分
  健康区间(40-60): +2分
  超买(>70): -3分

+/- MACD影响
  金叉: +5分
  死叉: -5分

+/- ROE影响
  >20%: +5分
  >15%: +3分
  <10%: -3分

+/- PE影响
  <15: +3分
  >50: -3分

最终分数 = clamp(0, 100, 基础分 + 各项调整)
```

### 操作建议
```
评分 >= 85: 强烈买入
评分 >= 70: 买入
评分 >= 55: 持有
评分 >= 40: 减持
评分 < 40:  卖出
```
